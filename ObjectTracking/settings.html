Image preprocessing

When enabled, apply a 3 × 3 Gaussian blur to the center crop before reference features are extracted. The blurred image is shown in the center view.

Crop size

Feature detector

Used after capturing a reference image. The top 100 features are drawn on the center view.

Tracking visualization

When tracking is active and a match is found, warp the live top-view frame into the reference image space using the inverse homography and show that projected image in the center view. When no projection is possible, show a gray surface.
