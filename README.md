## [Image Capture / Image Data Collection](image_capture/image_capture.html)

Capture sharp, labeled photos at full camera resolution.

- Save images with custom filenames so you can identify different items and item classes.
- Crop to the center of the frame during capture, keeping only the important detail.
- Use the live focus score to see when you are too close and the image would be blurry.
